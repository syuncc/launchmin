# Launchmin

An open-source admin management system.

> **⚠️ This project is currently under active development and is not ready for production use.**

## Design Philosophy

- **Security First** — Security is a foundational requirement, not an afterthought
- **Out of the Box** — Minimal configuration to get up and running
- **Accessible** — WCAG AA compliance as a non-negotiable standard
- **User Experience** — Every interaction is intentional, with clear feedback
- **Minimalism** — Only what matters, nothing more

## Tech Stack

- **Frontend:** React, MUI, Vite, TypeScript
- **Backend:** Hono, Node.js, TypeScript
- **Database:** MongoDB
